# antd-pro-toolkit
🐜 ant design pro toolkit is an advance common components library.

## Usage

```bash
npm install --save antd-pro-toolkit
```

```javascript
import { SlideVerify } from 'antd-pro-toolkit';
```

## Dependency

- antd
- lodash
